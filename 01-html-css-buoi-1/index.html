Giới thiệu Buổi 1

Buổi 1 tập trung vào cấu trúc HTML (Semantic tags) và CSS Box Model (margin, padding, border).

Mục tiêu: Tạo được một trang web đơn giản với layout rõ ràng, dễ đọc, sử dụng các thẻ semantic như <header>, <main>, <section>, <footer>.

Nội dung chính

Ví dụ Box Model

Khối màu bên dưới đang áp dụng padding, border và margin. Hãy mở file styles.css để chỉnh các giá trị và quan sát sự thay đổi.

Đây là một khối box model. Thay đổi margin, padding, border trong styles.css để hiểu rõ hơn.

Bài tập gợi ý

  1. Tạo thêm một section mới mô phỏng trang chủ Google:
    • Logo (có thể dùng chữ thay cho ảnh).
    • Ô tìm kiếm (dùng <input type="text">).
    • 2 nút bấm dưới ô tìm kiếm.
  2. Thử thay đổi các giá trị margin, padding, border của .demo-box trong styles.css.
  3. Thêm phần footer hiển thị tên bạn và câu: "Đã hoàn thành Buổi 1".
