## Buổi 1 - HTML5 Semantic & CSS Box Model

Thư mục này dùng để luyện tập **Buổi 1** trong file `Manh - FE timeline.txt`:
- Cấu trúc HTML (Semantic tags)
- CSS Box Model (margin, padding, border)

### Cách sử dụng

1. Mở file `index.html` bằng trình duyệt (Chrome, Edge, Firefox,...).
2. Mở file `styles.css` bằng VSCode/Cursor để chỉnh sửa.
3. Thử thay đổi các giá trị:
   - `margin`
   - `padding`
   - `border`

và quan sát sự thay đổi của khối `.demo-box`.

### Gợi ý bài tập thêm

- Thử clone lại giao diện trang chủ Google (bản đơn giản) trong chính file `index.html`:
  - Logo text "Google"
  - Ô tìm kiếm
  - 2 nút bên dưới ô tìm kiếm
- Dùng thêm các thẻ semantic:
  - `header`, `nav`, `main`, `section`, `article`, `footer`
- Thêm một đoạn mô tả có chứa từ khóa mà bạn muốn SEO (ví dụ: "Học HTML CSS cơ bản").
